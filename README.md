# Grade Clicker

Приложение-кликер для набора баллов. За каждый клик +5 баллов. При достижении 50, 70, 90, 100 баллов меняется картинка.

---

## Цель работы

Изучить жизненный цикл Activity (onCreate, onStart, onResume, onPause, onStop, onRestart, onDestroy) и научиться сохранять данные при повороте экрана.

---

## Контрольные вопросы

**1. Какие три метода вызываются при запуске приложения?**
onCreate → onStart → onResume

**2. Чем отличается onPause() от onStop()?**
onPause() — Activity теряет фокус, но ещё видна. onStop() — Activity полностью невидима.

**3. Когда вызывается onRestart()?**
При возврате в приложение после onStop() (например, нажали Home и вернулись).

**4. Что происходит при повороте экрана?**
Activity уничтожается и создаётся заново. Данные теряются, если их не сохранить.

**5. В чём разница между remember и rememberSaveable?**
remember сохраняет данные только при рекомпозиции. rememberSaveable сохраняет ещё и при повороте экрана.

**6. Почему onCreate() должен вызывать super.onCreate()?**
Чтобы родительский класс (ComponentActivity) правильно инициализировал Activity.

**7. Для чего используется класс Log?**
Для вывода сообщений в Logcat (отладка приложения).

**8. Какой метод вызывается только один раз за время жизни Activity?**
onCreate()
